A plugin to manage oil consumption based on burner operating hours. Allows users to enter multiple consumption records per year and view them in a table.

== Description ==

This plugin allows users to manage their oil consumption by tracking burner operating hours. It includes:
- A custom post type for consumption entries.
- ACF integration for detailed record-keeping.
- A dashboard widget that shows users their data and, for admins, the data of all users.

== Installation ==

1. Download the plugin.
2. Upload the plugin files to the `/wp-content/plugins/cost-sharing-burner-hours/` directory.
3. Activate the plugin through the 'Plugins' screen in WordPress.
